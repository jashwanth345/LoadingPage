# Loading Page Project

A simple Spring Boot web application that displays a responsive loading page using HTML and CSS.

## 🚀 Technologies Used
- Java 17
- Spring Boot
- Maven
- HTML5
- CSS3

## 📂 Project Structure

src/main/java - Java source files

src/main/resources/static - HTML and CSS files

src/main/resources/application.properties - Application configuration

## ✨ Features
- Responsive loading page
- Clean user interface
- Spring Boot backend integration
- Easy to customize and extend

## ▶️ How to Run

1. Clone the repository
2. Open the project in Spring Tool Suite (STS) or Eclipse
3. Update Maven dependencies
4. Run `LoadingPageApplication.java`
5. Open browser and visit:

http://localhost:8080/
